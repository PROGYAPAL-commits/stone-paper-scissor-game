# 🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer and enjoy a clean, responsive interface.

## 🚀 Live Demo
https://stone-paper-scissor-game-psi-bay.vercel.app

## 📌 Features
- Play Rock, Paper, Scissors against the computer
- Random computer move generation
- Live score tracking
- Responsive and user-friendly interface
- Instant game result display

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```
stone-paper-scissor-game/
│── images/
│── index.html
│── style.css
│── app.js
```

## 💻 How to Run
1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
